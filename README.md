# 🔢 Scientific Calculator (Web App)

A clean and functional scientific calculator built using **HTML, CSS, and JavaScript**.

This project was developed as part of a **5-day coding challenge**, focusing on improving frontend skills and GitHub workflow.

---

## 🚀 Features

- Basic operations: ➕ ➖ ✖️ ➗
- Scientific functions:
  - Square (x²)
  - Square root (√)
  - Power (xʸ)
  - Percentage (%)
- Keyboard support:
  - Numbers and operators
  - Enter (=)
  - Backspace (delete)
  - Escape (clear)
- Error handling:
  - Displays "Error" for invalid expressions
  - Handles division by zero
- Clean and responsive UI
- Styled operator buttons (pink theme 💗)

---

## 🛠️ Technologies Used

- HTML
- CSS (Grid + Styling)
- JavaScript (DOM + Logic)
- Git & GitHub

---


## 🧪 How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
